Optional

Install the ConnectWise Mobile App

The ConnectWise Mobile add-on gives you the power of ConnectWise anytime, anywhere so you can have instant access to contacts, tickets, and configurations.

Prerequisites

  • To purchase a mobile license, contact Sales@ConnectWise.com if you have not already done so.
  • Check out the Supported Versions page for accepted phones and operating systems.
  • You must perform the following steps in ConnectWise before you download the mobile app:
    • Navigate to System > Members > drill into a Member.
    • Select the Enable Mobile Edition check box from the Member Information section.
    • Repeat the steps above for each Member that will be using the mobile app.
Install

Android

Adding the ConnectWise Mobile App

  • On your phone, navigate to the Google Play Store.
  • Search for ConnectWise Mobile, then click Install.
  • Tap the ConnectWise Mobile icon after it has installed and complete the following fields:
    • Site - This is your company established ConnectWise site name.
      • Note: Do not use "http://" or "https://" when entering your site.
    • Company - This is your company name.
    • Member - Your member ID.
    • Password - Application password.
    • Tap Login.

Install the ConnectWise ChatAssist Feature

ChatAssist allows you to easily chat on new and existing tickets with ConnectWise Support.

  • Click the Install button below.
  • Save the ConnectWiseSupport.msi file to your desktop.
  • Double-click on the ConnectWiseSupport.msi file to complete the installation.
  • Log in with your ConnectWise University credentials.
  • The client will run in the system tray and allow you to set your availability and view support tickets.
